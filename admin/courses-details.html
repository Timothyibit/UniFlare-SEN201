Introduction to Software Engineering (SEN 201)

This course provides students with foundational knowledge of software engineering principles, methodologies, and best practices. It covers the entire software development lifecycle (SDLC), from requirement gathering to system deployment and maintenance.

Overview

  • Lectures 8
  • Quizzes 1
  • Duration 1 Semester
  • Skill level 200L
  • Language English
  • Students 300
  • Assessments Yes
Course Description

Software Engineering is a systematic, disciplined, and measurable approach to the design, development, operation, and maintenance of software.

Certification

Students who successfully complete this course will receive a Certification Badge in Software Engineering Fundamentals from the University of Abuja. This badge signifies proficiency in software development methodologies, requirement analysis, process architecture, and software lifecycle management. It can be shared on professional platforms to showcase expertise in software engineering.

Learning Outcomes
  • Understand the definition and scope of software engineering.
  • Learn about software requirement gathering and analysis
  • Explore various software development models such as the Waterfall, Agile, and Spiral models.
  • Software Design and Development
  • Understand software process architectures and their significance.
  • Understand software process architectures and their significance.
  • Develop an understanding of software design, testing, and maintenance.
